# 产品判断原则

## 核心用户

目标用户是需要在真实外贸场景中开口沟通的人，尤其是电话跟进、展会接待、报价、订单推进和售后沟通场景中的外贸业务员。

## 当前价值

- 不是泛英语听力，而是外贸业务场景中的思考、表达和口语演练。
- 每个小节对应一个明确业务任务，用户练的是如何理解客户意图、组织回应并推动事情进入下一步。
- 复用已有外贸课程、精确字幕、原声、知识点、账号和会员能力，避免为验证新训练方式重做数据与商业化基础。
- 第一章免费试学，后续通过添加微信购买会员邀请码承接商业化；小程序内不接微信支付，会员权益为全部章节 1 年访问权。

## 稳定版与实验版

- `main` 保留已完成的 EnglishPod 外贸复刻版，继续提供经典 Shadow / 逐句跟读体验。
- `codex/scenario-speaking-coach` 验证“外贸场景口语教练”方向，不直接覆盖稳定版。
- 实验版默认进入“今日 / 场景 / 复习 / 我的”，经典首页和课程页仍可进入，便于同一份内容对比两种产品形态。

## 实验版训练闭环

1. 明确场景角色和本次业务目标。
2. 完整听一遍，观察双方如何推进沟通。
3. 先独立组织回应，再查看参考表达。
4. 逐句听原声、录下自己的表达并自评。
5. 连续跟完整场景，把需要巩固的表达带入复习队列。

第一版不做自动语音评分。没有可靠识别和评价依据时，明确的业务目标、原声对比和用户自评比一个不透明分数更可信。

## 当前阶段优先级

1. 验证用户是否愿意完成“先思考、再表达、再跟读”的完整场景训练，而不只是播放课程。
2. 保持小节播放范围和中英文时间准确，不能串场景。
3. 让上次训练位置、掌握状态和复习入口有清晰反馈。
4. 保持 `waimao-mini` 数据隔离，不影响 EnglishPod 和外贸 Web 版。
5. 用户确认方向后，再讨论语音识别评分、训练状态云同步、CDN 和 Web 账号打通。
