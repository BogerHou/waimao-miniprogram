# 产品判断原则

## 核心用户

目标用户是需要在真实外贸场景中开口沟通的人，尤其是电话跟进、展会接待、报价、订单推进和售后沟通场景中的外贸业务员。

## 当前价值

- 不是泛英语听力，而是外贸业务场景的影子跟读。
- 复用 EnglishPod 已验证的练习体验，降低重做播放和 AI 交互的风险。
- 用小节粒度降低学习门槛，每个小节对应一个明确业务场景。
- 第一章免费试学，后续通过添加微信购买会员邀请码承接商业化；小程序内不接微信支付，会员权益为全部章节 1 年访问权。

## 当前阶段优先级

1. 复刻完整、少改布局。
2. 数据隔离，不能影响 EnglishPod。
3. 小节播放范围准确，不能串场景。
4. 外贸知识点能在新页面直接展示。
5. 后续再优化 CDN 和 Web 账号打通。
