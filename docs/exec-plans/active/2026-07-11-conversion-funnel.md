# 免费到会员的转化链路补强

## 目标

修补试学到付费漏斗上的明显断点，让锁定内容更有说服力、购买路径更顺、分享能带来回流。

## 范围

- 包含：锁定小节内容预览、解锁成功后回到来源场景、购买路径"复制微信号"兜底、分享海报加小程序码、会员临期提醒。
- 不包含：小程序内支付（明确不做）、价格与营销文案调整。

## 背景

- 相关文档：`docs/PRODUCT_SENSE.md`（第一章免费 + 邀请码商业化）、`docs/FRONTEND.md` 解锁页边界。
- 相关代码路径：`miniprogram/pages/index/index.ts`（`goToDetail` 锁定即弹解锁）、`miniprogram/pages/unlock/unlock.ts`（`goHome` 回首页、二维码预览）、`miniprogram/utils/share-poster.ts`、`miniprogram/store/index.ts`（`entitlement.expiresAt`）。
- 已知约束：小程序码需后端 `wxacode.getUnlimited` 接口配合（englishpod-server 仓库）；锁定预览只读字幕，不能提供音频播放。

## 风险

- 风险：锁定预览若能播放音频等于免费放出内容。
- 缓解：预览仅渲染前 3 句字幕文本，播放控件隐藏，底部固定解锁引导。

## 里程碑

1. 锁定小节预览态 + 点击锁定的动线改造（预览页内引导解锁）。
2. 解锁页：带 `fromSceneId` 参数，兑换成功后返回来源场景；新增复制微信号按钮（微信号进 app config）。
3. 海报小程序码：后端出码接口 + 前端海报叠加，scene 参数直达对应小节。
4. 会员临期（7 天内）首页 hero 提示续费路径。

## 验证方式

- 命令：`npm run typecheck && npm test`（share-card/unlock-format 相关测试扩展）。
- 手工检查：未登录/已登录/已解锁三种状态下锁定小节的行为；扫海报码进入正确小节；临期提示出现与消失条件。
