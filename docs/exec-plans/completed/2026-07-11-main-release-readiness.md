# Main 版本上线准备

## 目标

在不合并 `codex/scenario-speaking-coach` 实验分支的前提下，以最新 `main` 为唯一发布源，完成外贸英语影子跟读小程序的上线前检查、必要修复、生产环境冒烟验证和发布资料收口，使项目达到可在微信开发者工具上传体验版并提交审核的状态。

## 范围

- 包含：小程序构建与测试、生产 API 与静态资源检查、微信项目配置与包体检查、首页/课程/知识点/登录/解锁关键流程验收、发布清单和历史记录。
- 包含：发现明确上线阻断问题时，直接在 `main` 做最小修复并重新验证。
- 不包含：实验版场景教练代码、微信支付、Web 账号打通、CDN 改造和产品结构重做。
- 不包含：未经用户确认直接提交微信审核或正式发布；上传体验版前先完成本地与线上门禁。

## 约束与风险

- 小程序继续只访问 `/api/waimao-mini/*`，不能读写 EnglishPod 用户、进度、邀请码或权益。
- 第一章匿名试学，后六章通过小程序专用邀请码解锁一年访问权。
- 微信登录、合法域名、隐私协议、类目和审核资料部分依赖微信公众平台配置，无法仅通过仓库代码证明。
- 线上后端与静态资源已在服务中使用，冒烟检查只做无副作用请求；不消费邀请码，不修改真实用户数据。

## 验证方式

- 自动检查：`npm run typecheck`、`npm test`、`git diff --check`。
- 生产冒烟：课程列表、课程详情、音频、购买二维码、交流群二维码、外贸品牌图标和未认证接口边界。
- 微信开发者工具：编译无错误，验证首页、第一章课程、知识点、锁定入口、登录面板、解锁页和固定导航。
- 发布配置：正式 appid、生产 API、包体、页面 JSON、后台音频声明和上传前版本说明。
