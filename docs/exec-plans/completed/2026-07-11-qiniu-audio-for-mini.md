# 小程序接入七牛私有音频

## 目标

让新版外贸英语小程序与 Web 版共用七牛私有音频，优先播放短期签名 CDN 地址，失败时自动回退现有服务器签名地址。

## 范围

- 后端 `/api/waimao-mini/courses/:id` 增加 `audioSources`，保持旧 `audio` 字段为服务器地址，兼容旧版小程序。
- 小程序课程详情读取真实音频源数组，按服务端配置选择七牛或服务器源。
- 回退依据从域名猜测改为明确的 `provider`，不改现有播放器和播放范围逻辑。
- 更新发布文档和生成配置；不操作微信开发者工具，真机验收由用户完成。

## 风险与约束

- 七牛域名必须加入微信 `downloadFile` 合法域名。
- 私有地址带有效期和签名，客户端不能缓存为永久地址。
- 服务端源必须始终保留，确保七牛配置或网络异常时仍可播放。

## 验证

- 后端 TypeScript 构建与外贸小程序接口相关测试。
- 小程序 TypeScript 检查与 `audio-source-fallback` 单测。
- 部署后仅检查课程详情返回 `qiniu -> server`，真机播放和弱网切换交给用户。
